## Example

### Problem

Print a right-angled triangle pattern.

### Output

```text
*
**
***
****
*****
```

### Python Solution

```python
n = 5

for i in range(n):
    for j in range(i + 1):
        print("*", end="")
    print()
```

### Complexity

* Time Complexity: O(n²)
* Space Complexity: O(1)
